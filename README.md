# Panacea  

**This game is available at [itch.io](https://tamefixation.itch.io/panacea).**  
**Here is the [Walkthrough](https://paste.plurk.com/show/QyUjqx7s4EXNoQJ9VaFm/).**  

以”Witches don’t burn”爲主題的game jam作品。  
「你」是一名有藥草知識的中世紀普通人，在口耳相傳之下有不少人會來拜訪你，只為求得一帖良方。這之中也有抱著其他意圖的人，你的名聲也招來了不好的事情。  
工藝／選擇導向／劇情  
遊玩時長：約5-10分鐘  
共四結局
